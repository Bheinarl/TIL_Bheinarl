# .   
    -  .  : 현재의 디렉토리
    -  ..  :  현재의 상위 디렉토리(부모 폴더)   
     
# touch
  - 파일 생성 명령어
  ```
  touch 파일이름.확장자
  ```

# <span style="color: #e5e514">~/xxx/xxx ()
  - 현재 내 위치 경로

# ls
  - 현재 디렉토리에 존재하는 파일 찾기
  - 현재 작업 중인 디렉토리 내부의 폴더 / 파일 검색   

# cd ./xxx/xxx
  - ./xxx/xxx 폴더로 디렉토리 이동
  - 현재 작업 중인 디렉토리를 변경 (위치 이동)
   
# mkdir
  - 새로운 디렉토리 생성

# start <file명>
  - <file명> 파일을 실행

# re(remove) <file명>
  - <file명> 파일을 삭제

# -xxx (옵션)
  - 필요에 따라 검색해서 사용
  - -r : 디렉토리 삭제
  - -al : all list
