## 문자열 조회 / 탐색 및 검증 Method
| 메서드 | 설명 |
|:-:|:-:|
| `str.find(x)` | x의 첫 번째 위치를 반환. <br />없으면 -1을 반환 |
| `str.index(x)` | x의 첫 번째 위치를 반환. <br />없으면 오류 발생 |
| `str.isupper()` | 대문자 여부 |
| `str.islower()` | 소문자 여부 |
| `str.isalpha()` | 알파벳 문자 여부 <br /> 단순 알파벳이 아닌 유니코드 상 Letter <br />(한국어도 포함)
