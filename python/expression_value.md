# Expression & Value


---
## 표현식 (Expression)
- 값으로 평가될 수 있는 코드 조각

## 값 (Value)
- 표현식이 평가된 결과

## 평가 (Evaluate)
- 표현식을 실행하여 값을 얻는 과정
- 표현식을 순차적으로 평가하여 프로그램의 동작을 결정

## 문장 (Statement)
- 실행 가능한 동작을 기술하는 코드 (조건문, 반복문 , 함수 정의 등)

## 타입 (Type)
- 변수나 값이 가질 수 있는 데이터의 종류를 의미
- 어떤 종류의 데이터인지, 어떻게 해석되고 처리되어야 하는지를 정의
- 타입은 2가지 요소로 이루어짐
  - '값'과 '값에 적용할 수 있는 연산 (연산자)'
