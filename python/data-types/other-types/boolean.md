## Boolean(불리언)
- 참(True)과 거짓(False)을 표현하는 자료형

## Boolean 표현
- 비교 / 논리 연산의 평가 결과로 사용됨

- 주로 조건 / 반복문과 함께 사용

- True와 False 첫 글자는 무조건 대문자

```python
bool_1 = True
bool_2 = False

print(bool_1) # True
print(bool_2) # False
print(3 > 1) # True
print('3' != 3) # True
```
